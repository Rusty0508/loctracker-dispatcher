## 🚀 Quick Start - Running in Minutes

### Option 1: Cloud Deployment (Recommended)
```bash
1. Fork this repository
2. Add your GPS tracker API credentials
3. System starts automatically
4. Access your dashboard immediately
```

### Option 2: On-Premise Installation
```bash
# Clone the system
git clone https://github.com/Rusty0508/loctracker-dispatcher.git

# Install dependencies
cd loctracker-dispatcher
npm install
pip install -r requirements.txt

# Configure your GPS provider
cp .env.example .env
# Edit .env with your credentials

# Launch the platform
npm run start
```
